BudgetWise is a modern personal finance web application that enables users to track expenses, manage monthly budgets, set savings goals, and visualize spending patterns using dynamic charts and insights.

---

## Features

### 1. Expense & Income Tracking
- Add, edit, and delete expenses
- Record income entries
- Automatic monthly totals
- Stored securely in LocalStorage

### 2. Interactive Dashboard & Visualizations
- Real-time donut charts for category-wise spending
- Line chart for monthly expenses
- Category trend chart for spending growth/decline
- KPI summary cards:
  - Total expenses
  - Budget usage percentage
  - Number of active savings goals

### 3. Budgets
- Set category-specific monthly budgets
- System alerts when spending approaches or exceeds budget
- Alerts for:
  - 80% budget consumed
  - Budget fully exceeded

### 4. Savings Goals
- Create and manage personal financial goals
- Visual progress tracking via progress bars

### 5. Smart Financial Insights
- Intelligent suggestions based on:
  - Spending patterns
  - Overspending categories
  - Budget usage
- Ready for AI/Gemini integration

### 6. Multi-Currency Ready
- INR default system
- Structure ready for:
  - USD
  - EUR
  - GBP
- Optional add-on: Real-time conversion API

### 7. Export Data
- Export all transactions to CSV
- Future-ready for:
  - PDF reports
  - Cloud backup

### 8. User Authentication (Firebase)
- Email & Password registration
- Secure login session
- Protected routes using React Router
- `onAuthStateChanged` session detection
- Redirects unauthorized users to login
